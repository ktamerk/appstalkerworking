About Appstalker

Appstalker is a cross-platform mobile social networking application that allows users to share which applications they have installed on their phones with their followers. Built with privacy and real-time features in mind, Appstalker brings transparency to app usage and helps discover new applications through your social network.

Key Features

  • User authentication and profile management
  • Share installed applications with followers
  • Follow/unfollow users and send friend requests
  • Like profiles and discover new users
  • Real-time notifications via WebSocket
  • Privacy controls for app visibility
  • Cross-platform support (iOS & Android)

API Endpoints

Authentication

POST /api/auth/register

Register a new user account

POST /api/auth/login

Login and receive JWT token

Profile Management

GET /api/profile/me

Get current user profile

GET /api/profile/:username

Get user profile by username

PUT /api/profile/me

Update profile information

GET /api/profile/search/:query

Search for users

Installed Apps

POST /api/apps/sync

Sync installed apps from device

GET /api/apps/me

Get my installed apps

PUT /api/apps/:appId/visibility

Update app visibility

Social Features

POST /api/social/follow/:userId

Follow a user

DELETE /api/social/follow/:userId

Unfollow a user

GET /api/social/followers

Get followers list

GET /api/social/following

Get following list

POST /api/social/friend-request/:userId

Send friend request

PUT /api/social/friend-request/:id

Accept/reject friend request

POST /api/social/like/:profileId

Like a profile

DELETE /api/social/like/:profileId

Unlike a profile

Notifications

GET /api/notifications

Get all notifications

PUT /api/notifications/:id/read

Mark notification as read

PUT /api/notifications/read-all

Mark all as read

Mobile App

The React Native mobile application is located in the /mobile directory. It includes all screens, navigation, API integration, and native modules for accessing installed apps on Android and iOS devices.
